# 启动说明

## 方式一：使用PowerShell脚本（推荐）

直接运行PowerShell脚本：

```powershell
.\start_all.ps1
```

这个脚本会：
- 同时启动后端和前端服务
- 显示所有服务的访问地址
- 按 Ctrl+C 可以停止所有服务

## 方式二：使用批处理脚本

```cmd
start_all.bat
```

这个脚本会：
- 在独立的命令窗口中启动后端和前端
- 显示所有服务的访问地址

## 方式三：手动启动

### 启动后端

```cmd
start_backend.bat
```

或

```cmd
cd backend
python -m uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

### 启动前端

```cmd
start_frontend.bat
```

或

```cmd
cd frontend
npm run dev
```

## 服务地址

启动成功后，可以通过以下地址访问：

- **前端应用**: http://localhost:5173
- **后端API**: http://localhost:8000
- **API文档**: http://localhost:8000/docs
- **API文档(ReDoc)**: http://localhost:8000/redoc

## 注意事项

1. 首次运行需要安装依赖：
   - 后端：`cd backend && pip install -r requirements.txt`
   - 前端：`cd frontend && npm install`

2. 如果前端端口被占用，Vite会自动使用其他端口（如5174、5175等）

3. 停止服务：
   - PowerShell脚本：按 Ctrl+C
   - 批处理脚本：关闭各个命令窗口
   - 手动启动：在对应的终端按 Ctrl+C

4. 确保Python和Node.js已安装并配置好环境变量
